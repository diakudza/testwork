Развернуть сервис можно двумя способами.

1. Локальный сервер - положить содержимое каталога в www каталог вашего веб сервера, а так же импортировать дамп базы
   данных.
2. Докер (на нем я разрабатывал) - перейти в корневой каталог сервиса и выполнить поочередно:
   `docker-compose build`,`docker-compose up -d`
   1. После поднятия контейнеров необходимо импортировать дамп базы. 
   2. Так же необходимо установить все зависимости композерe.
   3. Для остановки контейнеров `docker-compose down`
Настройки приложения лежат в корневом каталоге , в файле `.env`
